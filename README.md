# Mod Changes:
- Gain 10 Boosts per Breed Cycle instead of 0.1
- Make levelling linear. 1 XP = 1 Level
- Increase Level Cap from 99 to 999
- Offspring are always better than or equal to parents in all stats
- Variance Cap of Stat Mutation increased by 10x, from ~2% variance to ~20% variance
